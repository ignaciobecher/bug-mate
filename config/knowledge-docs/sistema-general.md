# Manual General del Sistema

## Acceso al sistema

El sistema se accede desde cualquier navegador web moderno (Chrome, Firefox, Edge).
Se recomienda usar Google Chrome actualizado para la mejor experiencia.

## Módulos disponibles

### Facturación
Permite crear, editar y anular facturas. Para crear una factura nueva:
1. Ir al menú Facturación > Nueva Factura
2. Seleccionar el cliente
3. Agregar los ítems con precio y cantidad
4. Presionar Confirmar

### Reportes
Genera informes en PDF o Excel sobre ventas, stock e ingresos.
Para exportar un reporte: Reportes > Seleccionar tipo > Rango de fechas > Exportar.

## Preguntas frecuentes

**¿Con qué frecuencia se hacen backups?**
El sistema realiza backups automáticos diariamente a las 3 AM.

**¿Puedo acceder desde el celular?**
Sí, el sistema es responsive y funciona desde el navegador del celular.

**¿Cuántos usuarios pueden usar el sistema al mismo tiempo?**
No hay límite de usuarios simultáneos.
